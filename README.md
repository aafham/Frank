# FRNK+

Premium contemporary fashion storefront built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, Motion, and Lucide icons.

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Vercel Deployment

This repository is configured for Vercel zero-configuration deployment with the Next.js App Router.

- Production deploys are triggered by pushes to `main` after the GitHub repository is connected in Vercel.
- Preview deployments are triggered by pull requests and non-production branches.
- Runtime config lives in `vercel.json`.
- Public environment variables should be configured in Vercel Project Settings. Start from `.env.example`.

Recommended Vercel environment variable:

```bash
NEXT_PUBLIC_SITE_URL=https://your-production-domain.example
```

## Production Checks

Run before deployment:

```bash
npm ci
npm run lint
npm run build
```

## Notes

- Homepage is an editorial commerce prototype with sticky navigation, mobile menu, quick view, search, wishlist, and cart drawer interactions.
- Generated campaign assets live in `public/images`.
- The app uses the Next.js App Router under `src/app`.
- Sitemap, robots.txt, Open Graph metadata, Twitter metadata, manifest, and PWA service worker support are generated by the app.
